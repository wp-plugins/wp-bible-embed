This plugin embeds a KJV Bible into any page or post using the [bible] shortcode.

== Description ==

While there are many Bible verse plugins, there are none that display the whole Bible... Until now. Display this KJV Bible on one of your pages with the [bible] shortcode.

== Installation ==

1. Upload `bible_main.php.` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the [bible] shortcode in any post or page
